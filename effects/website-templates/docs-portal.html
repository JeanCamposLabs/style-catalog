Getting started / Quickstart

Quickstart

Get from zero to your first deploy in under five minutes. This guide walks through installing the CLI, authenticating, and shipping a project.

Install the CLI

Install globally with your package manager of choice:

bash
# install
npm i -g @nimbus/cli
💡
Tip: Already have the CLI? Run nimbus upgrade to get the latest.

Authenticate

Log in to link the CLI to your account. A browser window will open to complete the flow.

bash
nimbus login

Deploy your project

From your project root, run a single command. Nimbus detects your framework, builds, and ships to the edge.

ts
await nimbus.deploy({ env: 'production' })

That's it — your app is live. Next, explore webhooks or the REST reference.
